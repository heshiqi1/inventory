## 核心文件

### 📄 forex_monitor_feishu.py
**类型**：主程序  
**作用**：外汇信号监控器核心代码  
**语言**：Python  
**行数**：512 行  
**依赖**：requests, pandas, numpy

**主要功能**：
- 从 Stooq 加载外汇/贵金属数据
- 计算技术指标（EMA、RSI、ATR）
- 识别 H1/H2 交易形态
- 推送信号到企业微信

**配置项**（第 25-47 行）：
```python
WECOM_WEBHOOK_URL     # 企业微信 Webhook 地址
SYMBOLS               # 监控品种列表
SCAN_INTERVAL_SECONDS # 扫描间隔（秒）
LOOKBACK_DAYS         # 历史数据天数
SIGNAL_FRESH_DAYS     # 信号新鲜度（天）
SIGNAL_CACHE_FILE     # 缓存文件路径
```

**使用方法**：
```bash
python forex_monitor_feishu.py
```

**停止方法**：
按 `Ctrl+C`

### 📄 sent_signals.json
**类型**：数据文件  
**作用**：缓存已发送的信号，防止重复推送  
**格式**：JSON  
**自动生成**：首次发现信号时创建

**示例内容**：
```json
[
  "EURUSD|long|2026-03-01",
  "XAUUSD|long|2026-03-02",
  "GBPUSD|short|2026-03-02"
]
```

**维护规则**：
- 自动保留最近 30 天的记录
- 可手动删除以重置缓存
- 不建议手动编辑

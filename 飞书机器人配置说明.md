## 🚀 快速开始

### 第一步：获取飞书 Webhook 地址

1. 打开飞书群聊（建议创建专门的监控群）
2. 点击群设置 → **群机器人** → **添加机器人**
3. 选择 **自定义机器人**
4. 设置机器人名称（例如："外汇信号监控"）
5. 复制生成的 **Webhook 地址**

Webhook 格式示例：
```
https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

### 第二步：配置脚本

打开 `forex_monitor_feishu.py`，找到第 30 行：

```python
FEISHU_WEBHOOK_URL = "YOUR_WEBHOOK_URL"
```

替换为你的 Webhook 地址：

```python
FEISHU_WEBHOOK_URL = "https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
```

### 第三步：运行监控器

```bash
python forex_monitor_feishu.py
```

## ⚠️ 常见问题

### Q1: 提示"飞书发送失败"？
**A:** 检查以下几点：
1. Webhook URL 是否正确复制
2. 网络连接是否正常
3. 机器人是否被移除

### Q2: 收不到消息？
**A:** 
1. 确认机器人在群聊中
2. 检查群聊通知设置
3. 查看控制台是否有"新信号"输出

### Q3: 消息格式显示异常？
**A:** 飞书卡片需要特定的 JSON 结构，脚本已经处理好，如果仍有问题，可能是飞书版本不兼容，请更新飞书客户端。

### Q4: 想同时保留 Telegram 和飞书？
**A:** 可以同时运行两个脚本：
```bash
# 终端1
python forex_monitor.py

# 终端2  
python forex_monitor_feishu.py
```

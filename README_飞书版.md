## ⚙️ 配置参数

### 必填参数

```python
FEISHU_WEBHOOK_URL = "YOUR_WEBHOOK_URL"  # ⚠️ 必须填写
```

### 可选参数

```python
# 监控品种
SYMBOLS = {
    'EURUSD': 'eurusd',
    'USDJPY': 'usdjpy',
    'GBPUSD': 'gbpusd',
    'XAUUSD': 'xauusd',
    'XAGUSD': 'xagusd',
}

# 扫描频率
SCAN_INTERVAL_SECONDS = 3600    # 1小时 = 3600秒

# 信号新鲜度
SIGNAL_FRESH_DAYS = 2           # 只推送2天内的信号

# 历史数据量
LOOKBACK_DAYS = 150             # 加载150天数据用于指标计算
```

## ⚠️ 注意事项

1. **Webhook 安全**
   - 不要公开分享 Webhook URL
   - 定期更换（如需要）
   - 删除机器人会使 URL 失效

2. **消息频率**
   - 飞书有频率限制（通常够用）
   - 默认1小时扫描一次
   - 避免设置过于频繁

3. **依赖包**
   ```bash
   pip install requests pandas numpy
   ```

4. **Python 版本**
   - 需要 Python 3.10+ (使用了 `|` 类型注解)
   - 或修改代码兼容旧版本

## 🐛 故障排查

### 问题：收不到消息

**检查清单：**
- [ ] Webhook URL 完整复制（包括 `https://`）
- [ ] 机器人在群聊中（未被移除）
- [ ] 网络连接正常
- [ ] Python 脚本无报错

### 问题：卡片显示异常

**可能原因：**
- 飞书客户端版本过旧 → 更新飞书
- JSON 格式错误 → 运行测试脚本验证
- 网络传输问题 → 重新发送

### 问题：ImportError

**解决方法：**
```bash
pip install requests pandas numpy
```

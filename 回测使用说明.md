# H1/H2 策略回测工具使用说明

## 📁 文件说明

### 核心脚本

1. **forex_monitor_wx.py** (实时监控)
   - 用途: 实时扫描外汇市场,发现信号推送企业微信通知
   - 数据源: Stooq 日线数据
   - 运行方式: `python forex_monitor_wx.py`
   - 适用场景: 7x24小时实时监控

2. **forex_backtest_1year.py** (最近1年回测)
   - 用途: 使用最近1年日线数据回测策略
   - 数据源: Stooq 日线数据
   - 运行方式: `python forex_backtest_1year.py`
   - 生成内容: 回测报告 + 交易图表
   - 适用场景: 验证策略在最近市场的表现

3. **forex_backtest.py** (长期回测)
   - 用途: 使用2022-2025年数据回测策略(3年)
   - 数据源: Stooq 日线数据
   - 运行方式: `python forex_backtest.py`
   - 适用场景: 长期策略验证

4. **forex_backtest_5min.py** (5分钟周期回测)
   - 用途: 尝试使用5分钟数据回测
   - 数据源: yfinance (受API限制,仅60天)
   - 运行方式: `python forex_backtest_5min.py`
   - 注意: 需要先安装 `pip install yfinance`
   - 限制: yfinance API频率限制,可能无法正常运行

### 关键指标说明

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| **胜率** | 盈利交易占比 | > 50% |
| **TP1触及率** | 达到第一止盈位的交易占比 | > 40% |
| **盈利因子** | 总盈利/总亏损 | > 1.5 |
| **收益率** | 相对初始资金的收益百分比 | > 10% |

## 📈 图表说明

### 图表组成部分

每张交易图表包含:

1. **上半部分 - K线图**
   - 红色K线: 阳线(收盘>开盘)
   - 绿色K线: 阴线(收盘<开盘)
   - 蓝色线: EMA20 (短期均线)
   - 橙色线: EMA50 (长期均线)
   - 红色虚线: 止损位

2. **标注**
   - 蓝色三角形↑: 多单入场
   - 蓝色倒三角形↓: 空单入场
   - 绿色/红色X: 离场位置
   - 浅蓝色框: 入场信息(信号类型、趋势、RSI、入场价)
   - 绿色/红色框: 离场信息(离场原因、价格、盈亏)

3. **下半部分 - RSI指标**
   - 紫色线: RSI(14)
   - 红色虚线: 超买线(70)
   - 绿色虚线: 超卖线(30)
   - 蓝色点: 入场时RSI
   - 绿色/红色点: 离场时RSI

### 离场原因说明

| 代码 | 说明 |
|------|------|
| **TP2** | 达到第二止盈位(2:1),全部出场 |
| **BE** | 触及保本止损(TP1后移动止损到入场价) |
| **SL** | 触及原始止损位 |
| **TIME** | 时间止损(持仓超过20根K线) |

## 💡 常见问题

### Q1: 为什么5分钟回测无法运行?
**A**: yfinance对5分钟数据有严格的API频率限制,且只能获取最近60天的数据。建议使用日线数据回测,或者等待一段时间后再尝试。

### Q2: 如何修改回测时间范围?
**A**: 编辑 `forex_backtest_1year.py`:
```python
# 修改这两行
END_DATE = datetime.now().strftime("%Y-%m-%d")
START_DATE = (datetime.now() - timedelta(days=365)).strftime("%Y-%m-%d")
# 改为自定义日期,例如:
START_DATE = "2024-01-01"
END_DATE = "2025-01-01"
```

### Q3: 如何只回测某个品种?
**A**: 编辑脚本中的 `SYMBOLS` 字典:
```python
SYMBOLS = {
    'XAUUSD': 'xauusd',   # 只测试黄金
}
```

### Q4: 如何调整风险比例?
**A**: 修改 `RISK_PER_TRADE`:
```python
RISK_PER_TRADE = 0.01  # 改为1%
RISK_PER_TRADE = 0.03  # 改为3%
```

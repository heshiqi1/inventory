# 🚀 快速开始 - H1/H2 外汇监控器（企业微信版）

## 三步快速启动

### 第一步：安装依赖（1分钟）

打开命令行，运行：

```bash
pip install requests pandas numpy
```

---

### 第二步：配置企业微信（2分钟）

#### 2.1 创建企业微信机器人

1. 打开企业微信，进入要接收通知的群聊
2. 点击群聊右上角的「**...**」（更多）
3. 选择「**添加群机器人**」
4. 点击「**新建一个机器人**」
5. 输入机器人名称，例如：**H1H2外汇信号**
6. 点击「**添加**」

#### 2.2 复制 Webhook 地址

创建成功后会显示 Webhook 地址，格式类似：

```
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=693a91f6-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

**⚠️ 重要**：完整复制这个地址（建议截图保存）

#### 2.3 填入脚本

1. 打开 `forex_monitor_feishu.py` 文件
2. 找到第 **32** 行：

```python
WECOM_WEBHOOK_URL = "YOUR_WEBHOOK_URL"
```

3. 替换为你刚复制的地址：

```python
WECOM_WEBHOOK_URL = "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的key"
```

4. 保存文件（**Ctrl+S**）

---

### 第三步：测试并启动（1分钟）

#### 3.1 测试配置

```bash
python test_wecom.py
```

**预期结果**：

## ⚙️ 常用自定义

### 只监控黄金和欧元

打开 `forex_monitor_feishu.py`，找到第 35-41 行，修改为：

```python
SYMBOLS = {
    'EURUSD': 'eurusd',
    'XAUUSD': 'xauusd',
}
```

### 改为 30 分钟扫描一次

修改第 43 行：

```python
SCAN_INTERVAL_SECONDS = 1800  # 30分钟 = 1800秒
```

### 只要今天的信号

修改第 45 行：

```python
SIGNAL_FRESH_DAYS = 1  # 只报告今天的信号
```

## ❓ 遇到问题？

### 收不到消息

1. 检查 Webhook 地址是否正确（完整复制）
2. 运行 `python test_wecom.py` 诊断
3. 查看控制台错误信息

### 数据加载失败

1. 检查网络连接
2. 确认能访问 stooq.com
3. 品种代码必须小写

### 其他问题

查看 `示例_企业微信配置.txt` 中的「常见错误处理」章节

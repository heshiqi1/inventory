# H1/H2 外汇实时信号监控器 - 企业微信版

## 📋 项目简介

这是一个自动化外汇交易信号监控系统，使用企业微信机器人推送实时交易信号。

### 核心特性

- ✅ **实时监控**：每小时自动扫描市场机会
- ✅ **智能策略**：双EMA排列 + RSI过滤 + H1/H2形态识别
- ✅ **即时通知**：发现新信号自动推送到企业微信群
- ✅ **防重复**：智能缓存机制，避免重复推送
- ✅ **免费数据源**：使用 Stooq 免费行情数据
- ✅ **多品种支持**：外汇、贵金属等多个品种

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install requests pandas numpy
```

### 2. 配置企业微信机器人

1. 在企业微信群中点击「...」→「添加群机器人」
2. 创建机器人并复制 Webhook 地址
3. 打开 `forex_monitor_feishu.py`，找到第 32 行：

```python
WECOM_WEBHOOK_URL = "YOUR_WEBHOOK_URL"
```

4. 替换为你的实际 Webhook 地址：

```python
WECOM_WEBHOOK_URL = "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的key"
```

### 3. 测试配置

```bash
python test_wecom.py
```

如果配置正确，企业微信群会收到测试消息。

### 4. 启动监控

```bash
python forex_monitor_feishu.py
```

## ⚙️ 自定义配置

### 修改扫描间隔

```python
SCAN_INTERVAL_SECONDS = 3600  # 默认1小时，可改为1800（30分钟）等
```

### 修改监控品种

```python
SYMBOLS = {
    'EURUSD': 'eurusd',  # 保留你想监控的品种
    'XAUUSD': 'xauusd',  # 删除不需要的行
}
```

### 修改信号新鲜度

```python
SIGNAL_FRESH_DAYS = 2  # 只推送最近N天的信号
```

## 🔍 策略说明

### 入场条件

1. **趋势确认**：双EMA排列（20日EMA和50日EMA）
2. **动量确认**：连续2根趋势K线或近3根收盘有方向性
3. **RSI过滤**：
   - 做多信号：RSI < 70（避免超买）
   - 做空信号：RSI > 30（避免超卖）
4. **形态识别**：H1/H2 或 L1/L2 突破形态

### 风险管理

- **止损**：基于形态低点/高点
- **止盈**：
  - TP1：1:1 风险收益比
  - TP2：2:1 风险收益比

## 🛠️ 故障排查

### 问题1：收不到消息

**检查项**：
- ✅ Webhook 地址是否正确
- ✅ 网络连接是否正常
- ✅ 机器人是否被移除
- ✅ 查看控制台错误信息

### 问题2：数据加载失败

**检查项**：
- ✅ 网络能否访问 stooq.com
- ✅ 品种代码是否正确（必须小写）
- ✅ 是否被防火墙拦截

### 问题3：重复推送信号

**解决方案**：
- 删除 `sent_signals.json` 文件重置缓存
- 检查 `SIGNAL_FRESH_DAYS` 配置

## 🔐 安全建议

1. **不要泄露 Webhook 地址**：任何人获得此地址都能向群发送消息
2. **定期更换机器人**：如果怀疑地址泄露，删除旧机器人重新创建
3. **专用群组**：建议在专门的交易信号群使用，避免干扰工作群

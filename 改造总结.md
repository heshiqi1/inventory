```python

```

## 🎯 改进亮点

### 1. 代码简洁性
**前**：需要判断 Webhook 类型，准备不同格式
```python
if is_flow_webhook(FEISHU_WEBHOOK_URL):
    message = format_signal_for_flow(name, signal, now_str)
else:
    message = format_signal_card(name, signal, now_str)
```

**后**：统一格式，直接调用
```python
message = format_signal_markdown(name, signal, now_str)
```

### 2. 配置简单性
**前**：需要区分飞书机器人和流程自动化
```python
# 可能是机器人: https://open.feishu.cn/open-apis/bot/v2/hook/...
# 可能是流程: https://www.feishu.cn/flow/api/trigger-webhook/...
```

**后**：统一格式
```python
# 只有一种: https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=...
```

### 3. 维护性
- 减少了 22% 的代码量
- 减少了函数数量（10 → 8）
- 统一了消息格式（Markdown）
- 简化了错误处理逻辑

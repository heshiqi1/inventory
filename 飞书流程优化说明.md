## 🆚 两种方式对比

### 流程自动化（当前）

**优点：**
- 灵活：可在流程中处理数据
- 可扩展：连接表格、审批等
- 可定制：自定义消息格式

**缺点：**
- 需要配置流程
- 不如卡片美观（除非自己格式化）

### 自定义机器人（推荐）

**优点：**
- 美观：原生交互式卡片
- 简单：无需配置流程
- 专业：颜色、图标、分栏

**缺点：**
- 功能单一：只能发消息

## 🔄 如何切换到自定义机器人

如果您想要更美观的卡片显示，可以切换：

### 步骤1：创建自定义机器人
1. 在飞书群中点击"设置" → "群机器人"
2. 选择"添加机器人" → "自定义机器人"（不是流程自动化）
3. 设置名称和头像
4. 复制 Webhook URL

### 步骤2：更新配置
打开 `forex_monitor_feishu.py`，修改第 32 行：

```python
FEISHU_WEBHOOK_URL = "https://open.feishu.cn/open-apis/bot/v2/hook/..."
```

**重要：** 新的 URL 格式必须是 `https://open.feishu.cn/...`

### 步骤3：重启监控器
代码会自动识别新的 Webhook 类型，发送美观的交互式卡片！

# 企业微信机器人配置说明

## 一、获取企业微信群机器人 Webhook 地址

### 1. 创建群机器人

1. 在企业微信 PC 端或手机端，进入需要接收通知的群聊
2. 点击群聊右上角的「...」(更多) 按钮
3. 选择「添加群机器人」
4. 点击「新建一个机器人」
5. 设置机器人名称，例如：「H1/H2外汇信号」
6. 点击「添加」

### 2. 获取 Webhook 地址

1. 创建成功后，会显示机器人的 Webhook 地址
2. 复制完整的 Webhook URL，格式类似：
   ```
   https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
   ```
3. 保存好这个地址（建议截图保存）

### 3. 配置到脚本中

打开 `forex_monitor_feishu.py` 文件，找到配置区：

```python
# ============================================================
# ★ 配置区 — 填入你的企业微信机器人 Webhook 地址后即可运行
# ============================================================
WECOM_WEBHOOK_URL = "YOUR_WEBHOOK_URL"   # 将这里替换为你的 Webhook 地址
```

将 `YOUR_WEBHOOK_URL` 替换为你复制的完整 Webhook 地址。

## 二、企业微信机器人 API 说明

### 消息类型

脚本支持两种消息格式：

#### 1. 文本消息（启动/停止通知）

```json
{
    "msgtype": "text",
    "text": {
        "content": "消息内容"
    }
}
```

#### 2. Markdown 消息（交易信号）

```json
{
    "msgtype": "markdown",
    "markdown": {
        "content": "Markdown 格式的内容"
    }
}
```

### Markdown 语法支持

企业微信支持的 Markdown 语法：

- 标题：`# 一级标题`、`## 二级标题`、`### 三级标题`
- 加粗：`**加粗文本**`
- 引用：`> 引用内容`
- 字体颜色：
  - `<font color="info">文本</font>` - 绿色
  - `<font color="comment">文本</font>` - 灰色
  - `<font color="warning">文本</font>` - 橙红色

### 返回结果

成功时：
```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

失败时：
```json
{
    "errcode": 错误码,
    "errmsg": "错误信息"
}
```

## 四、使用注意事项

### 1. 频率限制

企业微信群机器人有以下限制：
- 每个机器人每分钟最多发送 20 条消息
- 单条消息内容不能超过 4096 字节

脚本默认每小时扫描一次，远低于频率限制。

### 2. 安全建议

- **不要将 Webhook 地址泄露给他人**，任何人获得此地址都能向群内发送消息
- 如果地址泄露，可以在群机器人设置中删除旧机器人，重新创建新的
- 建议在专门的交易信号群中使用，避免干扰正常工作群

### 3. 测试机器人

配置完成后，可以先运行脚本测试：

```bash
python forex_monitor_feishu.py
```

启动后会立即发送一条启动通知到企业微信群，如果收到消息说明配置成功。

## 五、常见问题

### Q1: 发送消息失败，提示 "errcode": 93000

**原因**：IP 地址不在白名单中

**解决方案**：
1. 在企业微信管理后台，找到「应用管理」→「自建」
2. 找到你的机器人应用
3. 在「可信IP」中添加你的服务器 IP 地址

**注意**：如果是个人电脑运行，IP 可能经常变化，建议使用固定 IP 的服务器。

### Q2: Markdown 格式显示不正常

**原因**：企业微信对 Markdown 的支持有限

**解决方案**：
- 检查是否使用了不支持的语法
- 确保使用的颜色标签是 `info`、`comment`、`warning` 之一
- 避免使用图片、链接等复杂格式

### Q3: 消息发送成功但群里看不到

**原因**：可能是机器人被移除或 Webhook 地址失效

**解决方案**：
1. 检查群机器人列表中是否还有该机器人
2. 如果被移除，需要重新创建并更新 Webhook 地址

### Q4: 想要更改机器人头像或名称

**解决方案**：
1. 在群聊中找到机器人
2. 点击机器人头像进入详情
3. 可以修改名称和头像（需要群主或管理员权限）

## 七、进阶配置

### 监控多个交易品种

在脚本配置区修改 `SYMBOLS` 字典：

```python
SYMBOLS = {
    'EURUSD': 'eurusd',   # 欧元/美元
    'USDJPY': 'usdjpy',   # 美元/日元
    'GBPUSD': 'gbpusd',   # 英镑/美元
    'XAUUSD': 'xauusd',   # 黄金/美元
    'XAGUSD': 'xagusd',   # 白银/美元
}
```

### 调整扫描间隔

```python
SCAN_INTERVAL_SECONDS = 3600   # 3600秒 = 1小时
```

可以改为：
- `1800` - 30分钟
- `7200` - 2小时
- `300` - 5分钟（注意频率限制）

### 调整信号新鲜度

```python
SIGNAL_FRESH_DAYS = 2   # 只报告最近2天内的信号
```

可以根据需求调整为 1-7 天。

## 八、后台运行（服务器部署）

### Windows（使用 nohup 或任务计划程序）

使用任务计划程序：
1. 打开「任务计划程序」
2. 创建基本任务
3. 设置触发器为「计算机启动时」
4. 操作选择「启动程序」，选择 Python 和脚本路径
5. 完成设置

### Linux（使用 systemd）

创建服务文件 `/etc/systemd/system/forex-monitor.service`：

```ini
[Unit]
Description=H1/H2 Forex Monitor
After=network.target

[Service]
Type=simple
User=your_username
WorkingDirectory=/path/to/your/script
ExecStart=/usr/bin/python3 forex_monitor_feishu.py
Restart=always

[Install]
WantedBy=multi-user.target
```

启动服务：
```bash
sudo systemctl daemon-reload
sudo systemctl start forex-monitor
sudo systemctl enable forex-monitor  # 开机自启
```

查看日志：
```bash
sudo journalctl -u forex-monitor -f
```

## 🔍 核心改动详解

### 1. 消息格式变更

**飞书版本**（交互式卡片）：
- 多列布局
- 丰富颜色模板
- 交互式按钮
- 复杂的 JSON 结构

**企业微信版本**（Markdown）：
- 单列布局
- 3 种颜色标签
- 纯文本展示
- 简洁的 Markdown 格式

### 2. API 调用变更

**飞书 API**：
```python
{
    "msg_type": "interactive",
    "card": {...}
}
# 返回: {"code": 0, "msg": "success"}
```

**企业微信 API**：
```python
{
    "msgtype": "markdown",
    "markdown": {"content": "..."}
}
# 返回: {"errcode": 0, "errmsg": "ok"}
```

### 3. 配置简化

**飞书版本**：
- 需要判断机器人/流程类型
- 两种不同的消息格式
- 复杂的错误处理

**企业微信版本**：
- 统一的配置方式
- 单一的消息格式
- 简化的错误处理

   - 不支持多列布局
